# 5-Stage-Pipelined-MIPS-Processor
Developed a complete RTL implementation of a 32-bit 5-stage pipelined MIPS processor in Verilog HDL, modeling the classic Instruction Fetch (IF), Instruction Decode (ID), Execute (EX), Memory (MEM), and Write-Back (WB) pipeline architecture. The design incorporates a hardware forwarding unit for resolving Read-After-Write (RAW) data hazards, a hazard detection unit for automatic load-use hazard stalling and pipeline bubble insertion, and a phase-shifted write-back mechanism to eliminate register file timing conflicts. The processor was verified using Icarus Verilog and GTKWave, with a configurable forwarding enable switch to facilitate comparative performance and waveform analysis during simulation.
